# Flo:Be
플라워 카페 상품 주문 & 예약 서비스

<br>

## 프로젝트 소개
- 플라워 카페의 홍보, 판매, 클래스 예약을 통합한 홈페이지
- 운영 및 관리의 불편함 해소, 소비자의 접근성 증가, 꽃 시장의 발전 도모
- 4인으로 진행한 팀 프로젝트
  
<br>

## 사용 언어 및 도구
- 언어: Java, SQL, html, css, jsp, javascript    
- 도구: maven, erwin, jdbc, Oracle DB
- 기타: git, notion, discord
